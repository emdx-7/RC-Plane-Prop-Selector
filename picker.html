Propeller Candidate Picker

All 611 merged propellers. Filter/sort the table, tick up to 12 to select, set the controls, then press Build plots to draw a 2D slice (quantities vs RPM) for each selected prop. Sliders and toggles only take effect on the next Build. Empirical (UIUC) points are converted to the same dimensional units and overlaid as hollow markers. A name prefixed with ~ (e.g. pl_~4x3) is an approximate inches conversion of a millimeter-sized prop.

Filter box tip: a column filter is normally a text match, but if you start it with >, <, or = it becomes a numeric threshold on that column. Examples: >10, <=5, =4, >=12. Use it on numeric columns (diameter, pitch, RPM, Vmax, thrust, power) to keep only rows above/below/equal to a value; combine columns to narrow the set.
